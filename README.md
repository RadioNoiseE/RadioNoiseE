## R.N.E？
RadioNoiseE，启发来源于『某魔法的禁书目录』。<br/>
初三，马上就高一。<br/>
语文一直不太好，对组版有超扭曲的执着。<br/>
青春期（aka，思春期☆）。<br/>
觉得在C里用`goto`有点宏语言的感觉，（自然）喜欢也滥用宏。<br/>
喜欢TeX丧心病狂的宏展开（当然仅限于某些特定时候）<br/>

不会写「自我介绍」（淦？）
